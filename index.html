Change boot disk in already patched iBECs for dualboot

This small tutorial is useful for people who already patched an iBEC for dual booting, but then need it again for a triple-multi boot

Also works for coolbooter iBECs.

Patching

Once you got your patched iBEC, open it in any HEX editor. I use HxD on windows and iHex on macOS.

You will see something like this:

Now search (ctrl+f on windows or cmd+f on mac) for disk0s1s3:

It will show you this:

Now replace disk0s1s3 with the disk you want to boot from. For example now I'll use disk0s1s5, meaning that my multi booted os is in the 5th partition. So my current setup is a triple boot.

Now iBEC will boot the system from disk0s1s5, but will still look for ramdisk (and other files needed for dualboot) in the old partition.

So, let's fix this. Search for upgrade

Make sure you found the "upgrade" near "failed to mount ... partition"

You will see a "2" near it. Replace it with the index number of the multibooted os partition. It is always a number minus the one you see in disk0s1sX. For example, in my case the boot partition is disk0s1s5, so the number will be 4.

Done! You can now boot the multibooted os using your freshly patched iBEC!
